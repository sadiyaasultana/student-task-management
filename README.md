# Student Task Management Web App

A simple and responsive **Student Task Management Web Application** that helps users add, complete, and delete daily tasks efficiently.  
Designed with a clean UI and local storage support to ensure tasks persist across sessions.

##  Features

- Add new tasks easily
- Mark tasks as **completed (✅)**
- Delete tasks when no longer needed
- Tasks stored using **Browser Local Storage**
- Clean and responsive UI
- Beginner-friendly and lightweight

##  Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Storage:** Browser Local Storage
- **IDE:** Visual Studio Code
- **Version Control:** Git & GitHub
